# UCSC Election Portal – Errors and Improvements

## Common Issues

1. **Error:** Website design looks simple and lacks modern styling.  
   **Change:** Add shadows, animations, and modern UI elements.

2. **Error:** Sidebar icons appear small and slightly inconsistent.  
   **Change:** Increase icon sizes and maintain consistent alignment.

3. **Error:** Large empty spaces appear in multiple pages.  
   **Change:** Add charts, tables, or information to fill spaces.

4. **Error:** Text sizes inconsistent across different pages.  
   **Change:** Maintain consistent typography sizes across all pages.


## Dashboard Section

1. **Error:** Dashboard contains large empty area below information.  
   **Change:** Add statistics cards or charts showing election progress.

2. **Error:** Cast vote button appears too wide visually.  
   **Change:** Reduce button width for better balanced layout.

3. **Error:** Election status card lacks additional details.  
   **Change:** Display election start time and remaining duration.


## Admin Dashboard Section

1. **Error:** Total students card does not show student number.  
   **Change:** Display total registered students inside statistics card.

2. **Error:** Live results section currently shows empty area.  
   **Change:** Add charts visualizing votes for each candidate.

3. **Error:** Reset and CSV buttons placed very close.  
   **Change:** Add spacing between control buttons for clarity.


## Cast Vote Section

1. **Error:** Candidate cards lack visible vote action button.  
   **Change:** Add clear vote button below candidate information.

2. **Error:** Voting timer text appears slightly small visually.  
   **Change:** Increase timer text size for better visibility.

3. **Error:** Page lacks confirmation popup before casting vote.  
   **Change:** Add confirmation dialog before submitting vote choice.


## Live Results Section

1. **Error:** Results page displays only progress bars currently.  
   **Change:** Add pie chart and bar chart visualizations.

2. **Error:** Candidate vote counts appear far from names.  
   **Change:** Align vote counts closer to candidate names.

3. **Error:** Results update message text appears too small.  
   **Change:** Increase message font size for better readability.
